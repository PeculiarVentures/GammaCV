# GammaCV Sample Application

#### Run development
```bash
npm run development
```
Visit [http://localhost:3000](http://localhost:3000)

#### Run production
```bash
npm run production
```
Visit [http://localhost:3000](http://localhost:3000)

### Build and deploy to gh-pages
```bash
npm run deploy
```

#### Run linter
```bash
npm run lint
```

#### Run tests
```bash
npm run test
```
For running tests in WebStorm add a Extra Mocha options `--compilers js:babel-core/register`
